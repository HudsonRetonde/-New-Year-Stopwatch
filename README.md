# -New-Year-Stopwatch

Cronometro que decrementa o tempo necessário para chegar ao próximo ano.

Projeto realizado com base na aula do Roger Melo no seu [canal do YouTube](https://www.youtube.com/watch?v=ms0mXabu5RY).
